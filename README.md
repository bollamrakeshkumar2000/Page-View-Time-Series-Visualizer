# Page View Time Series Visualizer

FreeCodeCamp Data Analysis with Python - Project 4

## Project Description
Time series analysis and visualization of freeCodeCamp forum page views from 2016-2019. This project demonstrates multiple visualization techniques for time series data.

## Visualizations Created
1. **Line Plot**: Daily page views trend over time
2. **Bar Plot**: Average monthly page views grouped by year
3. **Box Plots**: 
   - Year-wise distribution (trend analysis)
   - Month-wise distribution (seasonality analysis)

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
